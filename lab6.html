🔬 Лабораторная работа 6

Создание формы, валидация и дополнительные элементы

🎯 Цель работы

Изучить основные теги и атрибуты HTML для создания веб-форм, освоить различные типы полей ввода, научиться реализовывать клиентскую валидацию встроенными средствами HTML, а также использовать дополнительные элементы для повышения удобства и функциональности форм.

1. Создайте HTML-документ с базовым каркасом и подключите кодировку UTF-8.

2. Добавьте тег <form> с атрибутами method="post", action="#" (для эмуляции отправки) и укажите уникальный id или name.

3. Внутри формы создайте текстовое поле (<input type="text">) для ввода ФИО с атрибутом placeholder="Введите ваше ФИО".

4. Добавьте поле для email (type="email") с обязательным атрибутом required и плейсхолдером example@domain.com.

5. Добавьте поле для пароля (type="password") с атрибутом minlength="6" и required.

6. Создайте поле ввода номера телефона (type="tel") с атрибутом pattern="[0-9]{11}" и подсказкой title="Введите 11 цифр".

7. Добавьте числовое поле (type="number") для указания возраста с атрибутами min="1" max="120" и step="1".

8. Создайте поле для выбора даты (type="date").

9. Добавьте поле для ввода URL (type="url") с требованием обязательного заполнения.

10. Реализуйте поле для загрузки файла (type="file") с атрибутом accept="image/*" (только изображения).

11. Создайте группу радио-кнопок (type="radio") для выбора пола (мужской, женский, другой) с одинаковым атрибутом name="gender" и обязательным выбором (required).

12. Добавьте чекбоксы (type="checkbox") для выбора интересов (спорт, музыка, программирование), минимум 2 из них.

13. Создайте выпадающий список (<select> и <option>) для выбора города проживания.

14. Реализуйте список с множественным выбором (<select multiple size="3">).

15. Добавьте поле с автодополнением <input list="browsers"> и <datalist id="browsers"> с вариантами браузеров.

16. Создайте многострочное текстовое поле (<textarea>) для комментариев с атрибутами rows="4" cols="50" и максимальной длиной maxlength="200".

17. Сгруппируйте несколько полей в <fieldset> и добавьте заголовок <legend> (например, «Личные данные», «Дополнительная информация»).

18. Добавьте к некоторым полям всплывающие подсказки через атрибут title и невидимый для скринридеров текст (aria-label).

19. Реализуйте кнопку отправки (<input type="submit"> или <button type="submit">Отправить</button>) и кнопку сброса (<input type="reset">).

20. Проверьте работу встроенной валидации HTML:

  • пустые обязательные поля;
  • некорректный email/URL;
  • число вне диапазона;
  • короткий пароль;
  • неподходящий формат телефона.

🎯 Результат работы

Разработана и отлажена HTML-страница, содержащая форму с различными типами полей (текст, пароль, email, телефон, число, дата, радиокнопки, чекбоксы, выпадающие списки, текстовое поле, загрузка файлов). Форма реализует встроенную HTML-валидацию (обязательные поля, проверка формата email, диапазона чисел, регулярные выражения) и содержит дополнительные элементы. Страница корректно отображается в современных браузерах, а данные формы могут быть отправлены на сервер.
