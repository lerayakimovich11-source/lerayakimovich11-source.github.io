🔬 Лабораторная работа 2

Навигация и ссылки

🎯 Цель работы

Изучить механизмы создания гиперссылок в HTML, освоить типы ссылок (абсолютные, относительные, якорные) и получить практические навыки организации навигации между несколькими веб-страницами, а также внутри одной страницы.

1. Простая внешняя ссылка

Создайте страницу index.html. Добавьте ссылку на сайт yandex.ru. Рядом со ссылкой напишите пояснение: «Спецификация HTML (откроется в этом же окне)».

2. Открытие ссылки в новой вкладке

Скопируйте ссылку из задания 1, но добавьте атрибут target="_blank". Добавьте ещё две любые внешние ссылки (например, на Google и ВКонтакте) с тем же атрибутом. Под каждой ссылкой укажите, куда она ведёт.

3. Атрибут title для всплывающей подсказки

Создайте ссылку с текстом «Узнать погоду». Укажите атрибут title="Прогноз на сегодня". При наведении курсора должна появляться подсказка.

4. Ссылка на email

Создайте ссылку mailto:info@example.com с текстом «Написать письмо». Добавьте тему письма и тело через параметры: mailto:info@example.com?subject=Вопрос&body=Здравствуйте.

5. Ссылка на телефон

Создайте ссылку tel:+71234567890 с текстом «Позвонить нам». Объясните (текстом на странице), для каких устройств этот тип ссылок наиболее полезен.

6. Простой якорь внутри страницы

Создайте страницу с тремя разделами (О себе, Хобби, Образование). Вверху сделайте три ссылки, ведущие к этим разделам. Для каждого раздела задайте уникальный id.

7. Якорь с возвратом наверх

Создайте длинную страницу (5+ абзацев). В самом низу добавьте ссылку «Наверх», которая ведёт в начало страницы (можно использовать id="top" или href="#").

8. Плавная прокрутка (без JS — только HTML)

Объясните студентам, что без CSS/JS плавной прокрутки нет. Тем не менее задание: создайте страницу с оглавлением и якорями. Добавьте комментарий в HTML: «для плавной прокрутки нужно свойство CSS scroll-behavior: smooth».

9. Якорь на другой странице

Создайте два файла: page1.html и page2.html. В page2.html сделайте раздел с id="chapter3". В page1.html сделайте ссылку, которая ведёт прямо в этот раздел page2.html#chapter3.

10. Несколько якорей в одном документе

Создайте страницу с 5 рецептами (каждый рецепт — отдельный блок с id). Вверху сделайте навигационное меню из 5 ссылок. Проверьте, что клик по каждой ссылке приводит к нужному рецепту.

11. Ссылка в той же папке

Создайте папку site. Внутри создайте index.html и catalog.html. В index.html сделайте ссылку на catalog.html. В catalog.html сделайте обратную ссылку на index.html.

12. Переход в подпапку и из подпапки

Внутри папки site создайте подпапку products. Перенесите catalog.html в products. Исправьте ссылки:

  • из index.html в products/catalog.html
  • из catalog.html в ../index.html

13. Абсолютный путь для тестирования

На странице index.html пропишите абсолютный путь к какому-либо файлу на вашем компьютере (например, C:\Users\Public\file.txt или /home/user/file.txt). Объясните, почему такой путь не будет работать на другом компьютере.

14. Ссылка на скачивание .txt файла

Создайте текстовый файл note.txt с любым содержимым. На странице сделайте ссылку на этот файл с атрибутом download. Проверьте, что браузер предлагает скачать файл, а не открыть его.

15. Ссылка на PDF и изображение

Найдите (или создайте) небольшой PDF-файл и картинку (.jpg/.png). Сделайте две ссылки:

  • одна открывает PDF в браузере
  • вторая открывает картинку

Добавьте пояснение, отличается ли поведение в разных браузерах.

16. Ссылка на архив (.zip)

Создайте пустой архив archive.zip. Сделайте ссылку на его скачивание. Добавьте атрибут download и текст «Скачать архив (ZIP, 0 KB)».

17. Навигационное меню из трёх страниц

Создайте сайт из трёх страниц:

  • index.html (Главная)
  • about.html (О нас)
  • contacts.html (Контакты)

На каждой странице разместите одинаковое меню (ссылки на все три страницы). Убедитесь, что меню работает с любой страницы.

18. Активная ссылка (стилизация без CSS – только комментарий)

Добавьте в меню комментарий: «<!-- здесь позже будет CSS-класс .active для текущей страницы -->». Рядом напишите, как вы будете отличать текущую страницу визуально (например, жирным шрифтом).

19. Карта ссылок в подвале (footer)

В нижней части каждой страницы (из задания 17) создайте блок «Полезные ссылки»: ссылки на 3 внешних ресурса (например, на учебник по HTML, на валидатор W3C, на MDN). Все ссылки должны открываться в новой вкладке.

20. Мини-проект: сайт-визитка с навигацией

Создайте мини-сайт из 4 страниц:

  • Главная – приветствие и ссылки на остальные страницы.
  • О нас – короткий текст + якорная ссылка «К контактам» (ведёт на страницу Контакты).
  • Услуги – список услуг, каждая услуга – ссылка на внешний сайт с примерами (открывается в новой вкладке).
  • Контакты – email, телефон, а также ссылка на карту (например, на Яндекс.Карты).

В каждой странице должны быть:

  • единое навигационное меню,
  • ссылка «Наверх» внизу,
  • хотя бы одна внутренняя ссылка на другую страницу с якорем (например, с «О нас» на раздел «Адрес» в «Контактах»).

🎯 Результат работы

Создан набор взаимосвязанных HTML-документов, образующих единый веб-сайт с работающей системой навигации.
