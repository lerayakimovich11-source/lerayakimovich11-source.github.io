🔬 Лабораторная работа 5

Создание формы, регистрация и обратная связь

🎯 Цель работы

Изучить HTML-элементы для создания веб-форм, освоить различные типы полей ввода, научиться организовывать данные для отправки на сервер, а также получить практические навыки разработки форм регистрации пользователей и формы обратной связи с базовой валидацией.

1. Создать HTML-документ

Создайте файл forms.html. Укажите кодировку UTF-8, заголовок страницы «Формы регистрации и обратной связи».

2. Добавить структуру страницы

Используйте заголовки <h1> (для всей страницы), <h2> (для каждой формы), разделите формы горизонтальной линией <hr> или отдельными <section>.

3. Создать первую форму (регистрация)

Откройте тег <form> с атрибутами: method="POST", action="/register" (условно) и name="registration_form".

4. Сгруппировать личные данные

Внутри формы регистрации создайте <fieldset> с заголовком <legend> «Личная информация».

5. Добавить текстовые поля ФИО

Создайте поле «Имя» (type="text", required, placeholder="Введите имя"). Аналогично — «Фамилия». Для каждого поля используйте <label>.

6. Добавить поле email

Добавьте поле type="email", атрибут required, placeholder="example@mail.ru".

7. Добавить поле выбора пола

Используйте два переключателя (type="radio") с одинаковым name="gender" и значениями male, female. Третий вариант — «Другой» или «Не указано».

8. Добавить выпадающий список (страна/город)

Создайте <select> с атрибутом name="country" и 3–5 вариантами <option> (например, Россия, Беларусь, Казахстан, Другая).

9. Создать второй fieldset для учётных данных

Назовите <legend> «Данные для входа».

10. Добавить поле пароля и подтверждения пароля

Два поля type="password", одно с именем password, второе — password_confirm. Для первого укажите minlength="6" и required.

11. Добавить чекбокс согласия с правилами

Создайте type="checkbox", name="agree", required. Текст: «Я принимаю условия пользовательского соглашения».

12. Добавить кнопки отправки и сброса

Кнопка type="submit" с надписью «Зарегистрироваться» и type="reset" с надписью «Очистить форму».

13. Проверить работу первой формы

Открыть в браузере, попробовать ввести данные, проверить, что работает required и minlength. (Форма не уйдёт на сервер — это нормально для лабораторной.)

14. Создать вторую форму (обратная связь)

После <hr> добавьте вторую форму с именем feedback_form. Метод POST, условный адрес action="/feedback".

15. Добавить поле темы сообщения

Текстовое поле type="text" с name="subject", required, placeholder="Краткая тема".

16. Добавить поле email для ответа

Поле type="email", name="reply_email", required, placeholder="ваш email для ответа".

17. Добавить многострочное поле (текст сообщения)

Элемент <textarea> с name="message", rows="5", cols="40", placeholder="Опишите ваш вопрос или проблему...", атрибут required.

18. Добавить оценку удовлетворённости

Варианты: три переключателя (radio) с оценками: «Отлично», «Хорошо», «Удовлетворительно». Или ползунок (type="range").

19. Добавить поле для загрузки файла

Элемент type="file", name="screenshot", accept="image/*" (чтобы разрешить только изображения).

20. Оформить кнопку отправки формы обратной связи

Добавить кнопку type="submit" с текстом «Отправить сообщение». Убедиться, что форма имеет все подписи <label> и что при клике на подпись активируется соответствующее поле.

🎯 Результат работы

Разработана HTML-страница, содержащая две функциональные формы: форму регистрации и форму обратной связи. Формы корректно отображаются в браузере, содержат все изученные типы полей ввода, базовые проверки на стороне клиента и демонстрируют готовую структуру для последующей интеграции с серверной частью.
