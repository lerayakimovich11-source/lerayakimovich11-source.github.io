📘 Лекция 5: Таблицы

🎯 Цель лекции: Сформировать понимание принципов создания и форматирования таблиц в HTML, а также освоить основные теги и атрибуты для структурирования табличных данных и их корректного отображения на веб-страницах.

Структура таблицы в HTML

Структура таблицы в HTML включает использование определённых элементов и атрибутов, а также правила оформления. Таблица строится последовательно: таблица → строка → ячейка. Если нарушить порядок, браузер не покажет таблицу или покажет исходный код.

Некоторые теги, используемые для создания таблицы в HTML:

  • <table> — контейнер для всего содержимого таблицы.
  • <tr> (table row) — строка таблицы.
  • <td> (table data) — ячейка с данными. Помещается внутрь <tr>.
  • <th> (table header) — ячейка заголовка. Используется внутри тегов <thead> и <tr>.
  • <caption> — определяет заголовок таблицы, который будет размещён над таблицей. Используется внутри тега <table>.
  • <thead> — определяет заголовок таблицы. Может содержать одну или несколько строк, в которых используются теги <th> для определения заголовков столбцов.
  • <tbody> — определяет тело таблицы. Содержит одну или несколько строк, в которых могут использоваться теги <td> для определения содержимого ячеек.
  • <tfoot> — определяет нижний колонтитул таблицы. Может содержать одну или несколько строк, в которых могут использоваться теги <td> для определения содержимого ячеек.

Также для группировки строк в таблице используются теги <colgroup> и <col>.

Атрибуты

Некоторые атрибуты, которые применяются при создании таблицы в HTML:

  • colspan — указывает, сколько столбцов должна занимать ячейка. Значение по умолчанию — 1. Например, чтобы ячейка растянулась на 3 столбца, пишут: <td colspan="3">Эта ячейка занимает 3 столбца</td>.
  • rowspan — указывает, на сколько строк вниз должна растянуться ячейка. Например: <td rowspan="2">Эта ячейка занимает 2 строки</td>.
  • scope — в теге <th> указывает, что это заголовок столбца, а scope="row" — строки. Это повышает доступность, так как скринридер может не понять, к какому столбцу относится значение.

Оформление

Для стилизации таблицы в HTML используется CSS. Некоторые CSS-свойства, которые формируют фундамент стилизации:

  • border — задаёт границы таблицы и ячеек.
  • border-collapse — определяет, будут ли границы ячеек отображаться отдельно или сливаться. Рекомендуется использовать border-collapse: collapse для чистого вида.
  • width/height — устанавливает размеры таблицы или отдельных ячеек.
  • padding — внутренние отступы в ячейках.
  • text-align — выравнивание текста в ячейках.
  • background-color — цвет фона таблицы или ячеек.

Также для стилизации конкретных колонок можно использовать селектор: nth-child.

Ошибки

Некоторые ошибки, которых следует избегать при создании таблицы в HTML:

  • Размещение <tr> прямо в <td> — это приведёт к ошибке в документе.
  • Пропуск закрывающих тегов — всегда нужно закрывать теги <table>, <tr>, <th> и <td> для правильного отображения и валидности HTML.
  • Использование таблиц для вёрстки макета страницы — этот подход устарел, современные методы — Flexbox и Grid — обеспечивают лучшую гибкость и контроль.

Объединение ячеек в HTML

Объединение ячеек в HTML-таблицах осуществляется с помощью двух атрибутов: colspan (объединение по горизонтали) и rowspan (объединение по вертикали). Эти атрибуты применяются к тегам <td> и <th>.

Атрибут colspan

colspan позволяет объединить несколько ячеек в одной строке, создавая ячейку, которая занимает пространство нескольких столбцов. Значение атрибута указывает количество столбцов для объединения.

Синтаксис:

<td colspan="число">Содержимое ячейки</td>

Например, colspan="2" означает, что ячейка займёт два столбца.

Практические сценарии использования:

  • создание общих заголовков для нескольких колонок;
  • группировка связанных столбцов под общей категорией;
  • итоговые строки с охватом нескольких столбцов;
  • добавление примечаний в нижней части таблицы.

Важно: общее количество ячеек в каждой строке должно быть одинаковым. Если в одной строке больше ячеек, чем нужно для объединения, могут появиться пустые или «фантомные» ячейки.

Атрибут rowspan

rowspan работает с вертикальным объединением, позволяя создавать ячейки, которые охватывают несколько строк. Значение атрибута указывает, сколько строк должна занимать ячейка.

Синтаксис:

<td rowspan="число">Содержимое ячейки</td>

Например, rowspan="3" означает, что ячейка займёт три строки.

При вертикальном объединении нужно учитывать, что ячейка, растянутая по вертикали, вытеснит ячейки из ряда ниже. Чтобы таблица сохранила структуру, в строках ниже следует удалить по одной ячейке.

Комбинирование colspan и rowspan

Можно одновременно использовать оба атрибута в одной таблице, чтобы создавать сложные многоуровневые структуры для представления иерархических данных или сложных отношений. Например:

<table>
  <tr>
    <td rowspan="2" colspan="2">Объединённая ячейка</td>
    <td>Содержимое 1</td>
    <td>Содержимое 2</td>
  </tr>
  <tr>
    <td>Содержимое 3</td>
    <td>Содержимое 4</td>
  </tr>
</table>

Группировка разделов таблицы в HTML

В HTML для группировки разделов таблицы используются несколько тегов, которые позволяют структурировать таблицу и применять стили к группам столбцов.

Тег <colgroup>

Этот тег создаёт структурную группу столбцов, выделяя логически однородные ячейки. С его помощью можно применять стили к нескольким столбцам одновременно, вместо того чтобы повторять стили для каждой ячейки или строки.

Особенности использования:

  • <colgroup> должен быть дочерним элементом тега <table> и располагаться после <caption>, если он есть, но перед другими элементами таблицы (<thead>, <tr>, <td> и т. д.).
  • Внутри <colgroup> можно использовать тег <col>, который задаёт свойства для отдельных столбцов в группе.
  • У тега <colgroup> есть атрибут span, который определяет количество столбцов в группе.
  • Если в одной таблице используется несколько тегов <colgroup>, порядок их появления влияет на стиль таблицы.

Пример:

<table>
  <colgroup>
    <col span="2" style="background-color: #D6EEEE">
    <col span="3" style="background-color: pink">
  </colgroup>
  <tr>
    <th>MON</th>
    <th>TUE</th>
    <th>WED</th>
    <th>THU</th>
  </tr>
  <!-- Другие строки таблицы -->
</table>

Ограничения: к <colgroup> применимы только ограниченная выборка CSS-свойств (например, width, visibility, background, border).

Другие теги для группировки разделов таблицы

  • <thead> — используется для создания группы заголовков таблицы. Обычно это первая строка с заголовками столбцов. В <thead> для ячеек используется тег <th>.
  • <tbody> — группирует основное содержимое таблицы. Можно использовать несколько раз для разбиения контента на части с общей логикой.
  • <tfoot> — используется для группировки данных в нижней части таблицы (нижний колонтитул). Например, для представления данных о суммах или итогах.

Важно: семантические теги помогают улучшить доступность и понимание структуры таблицы как для пользователей, так и для поисковых систем.

Если требуется более сложная группировка или работа с несколькими колонками (не в рамках таблицы), можно использовать CSS Grid или другие методы разметки.

Связка заголовка с ячейкой в HTML-таблицах

Связка заголовка с ячейкой в HTML-таблицах обеспечивается с помощью элементов <th> (ячейки заголовка) и атрибутов, которые уточняют семантику и связь между элементами.

Тег <th>

Элемент <th> используется для создания заголовочной ячейки в таблице. Он семантически обозначает заголовок столбца или строки. По умолчанию текст в таких ячейках отображается жирным шрифтом и выравнивается по центру.

Синтаксис:

<tr>
  <th>Заголовок</th>
</tr>

Важно: тег <th> должен размещаться внутри контейнера <tr>, который, в свою очередь, располагается внутри тега <table>.

Атрибуты для связи заголовка с ячейкой

1. Атрибут scope. Указывает, к какой части таблицы относится заголовок. Возможные значения:

  • col — заголовок относится ко всем ячейкам в столбце.
  • row — заголовок относится ко всем ячейкам в строке.
  • colgroup — заголовок относится ко всем ячейкам в группе столбцов.
  • rowgroup — заголовок относится ко всем ячейкам в группе строк.

Пример:

<tr>
  <th scope="col">Месяц</th>
  <th scope="row">Дата</th>
</tr>

2. Атрибут headers. Позволяет связать ячейки таблицы с заголовками, особенно для пользователей речевых браузеров. Для этого одной ячейке задаётся атрибут id, а второй — атрибут headers со значением, совпадающим с id первой ячейки.

Пример:

<tr>
  <th id="c1">Number</th>
  <th id="c2">element</th>
</tr>
<tr>
  <td headers="c1">4.1.1</td>
  <td headers="c2">html</td>
</tr>
