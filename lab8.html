🔬 Лабораторная работа 8

Создание индивидуального сайта со всеми изученными функциями

🎯 Цель работы

Закрепить и систематизировать знания по основам HTML, полученные в ходе изучения предыдущих тем, путем разработки полноценного многостраничного сайта, объединяющего все изученные элементы (структура документа, работа с текстом, изображения, гиперссылки, таблицы, списки, формы).

1. Создать корневую папку сайта

Создайте папку my_site. Внутри неё создайте папки: images (для изображений), pages (для дополнительных страниц), css (для стилей).

2. Создать главную страницу index.html

Добавьте правильный DOCTYPE, теги <html>, <head>, <body>. Пропишите русскую кодировку (UTF-8) и заголовок окна браузера (тег <title>).

3. Добавить заголовки и подзаголовки

Используйте теги от <h1> до <h3> на главной странице. Например: <h1> — имя сайта, <h2> — название раздела, <h3> — подраздел.

4. Вставить абзацы с форматированием текста

Напишите приветственный текст (3–4 абзаца). Примените теги: <p>, <br>, <hr>. Выделите важные слова с помощью <strong>, <em>, <u>, <del>.

5. Добавить маркированный и нумерованный списки

Создайте блок «Мои навыки» (маркированный список) и «План развития» (нумерованный список).

6. Вставить изображения

Добавьте не менее 2 изображений (фото, иконка, логотип). Обязательно используйте атрибуты src, alt, width, height.

7. Создать навигационное меню

Вверху каждой страницы разместите ссылки на все разделы сайта (Главная, О себе, Таблица, Контакты, Галерея). Используйте тег <a> и относительные пути.

8. Создать страницу «О себе» (about.html)

Разместите в папке pages. Добавьте текст о себе, фото, списки увлечений и цитату (тег <blockquote>).

9. Создать страницу с таблицей (table.html)

Постройте таблицу «Расписание» или «Прайс-лист». Используйте border, cellpadding, cellspacing. Добавьте объединение ячеек (colspan, rowspan). Обязательно выделите заголовок таблицы с помощью <th>.

10. Создать страницу «Галерея» (gallery.html)

Разместите минимум 4 изображения с подписями под ними. Расположите их в строку или сеткой (можно с помощью таблицы или базового CSS).

11. Создать страницу с формой (form.html)

Реализуйте форму обратной связи или регистрации, содержащую:

  • текстовое поле;
  • поле email;
  • многострочное поле (<textarea>);
  • выпадающий список (<select>);
  • радиокнопки или чекбоксы;
  • кнопку отправки (<button type="submit">).

12. Добавить ссылки на внешние ресурсы

Сделайте блок «Полезные ссылки» (например, на справочник HTML или CSS). Ссылки должны открываться в новой вкладке (атрибут target="_blank").

13. Реализовать внутренние якоря

На длинной странице (например, «О себе») создайте содержание вверху и якоря (id) для быстрого перехода к разделам.

14. Вставить горизонтальные линии и специальные символы

Используйте <hr> для разделения блоков. Добавьте символы: копирайт &copy;, стрелку &rarr;, торговую марку &trade; и другие.

15. Добавить подвал (footer) на всех страницах

Внизу каждой страницы укажите: год создания, ваше имя/ник, иконку почты (можно текстом) и ссылку «Наверх» (якорь).

16. Стилизовать с помощью базового CSS

Создайте файл style.css в папке css и подключите его ко всем страницам. Пропишите минимум:

  • цвет фона страницы;
  • цвет и шрифт текста;
  • стиль ссылок (обычный, наведение);
  • рамки и отступы для таблицы;
  • выравнивание заголовков.

17. Обеспечить единый дизайн всех страниц

Используйте общий фон, одинаковое меню, одинаковый шрифт и цветовую гамму на всех страницах сайта.

18. Добавить комментарии в код HTML и CSS

Объясните сложные моменты (объединение ячеек, структуру формы, блок стилей) с помощью <!-- комментарий -->.

19. Проверить валидность и кросс-браузерность

Откройте сайт в трёх браузерах. Проверьте все ссылки, форму, отображение картинок. Исправьте ошибки кодировки, если русский текст отображается кракозябрами.

20. Создать архив и файл README.txt

Упакуйте папку my_site в ZIP-архив. Внутри архива или отдельно приложите текстовый файл с кратким описанием: тема сайта, какие функции реализованы, список страниц.

🎯 Результат работы

Разработан работающий сайт, состоящий минимум из 3–5 HTML-страниц, связанных между собой системой навигации, демонстрирующий грамотное использование всех изученных тегов и атрибутов. Сайт должен корректно отображаться в современных браузерах (Chrome, Firefox, Edge) и быть оформлен в едином стиле (цветовая гамма, шрифты, выравнивание).
